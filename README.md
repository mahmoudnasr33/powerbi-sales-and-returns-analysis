# Sales & Returns Dashboard (Power BI)

## Project Objective
To analyze retail sales and returns performance across stores, products, and months using an interactive Power BI dashboard.

## Dataset Used
- ## https://github.com/mahmoudnasr33/powerbi-sales-and-returns-analysis/blob/main/DA%20Project1.xlsx

## Questions (KPIs)

- What are the total sales, total returns, total quantity, and number of stores?
- Which months generate the highest sales?
- Which products generate the highest sales?
- Which stores have the highest and lowest sales performance?
- Which products record the highest and lowest returns?
- How do monthly filters affect sales and returns performance?
