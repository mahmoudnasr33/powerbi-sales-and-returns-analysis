# Sales & Returns Dashboard (Power BI)

## Project Objective
To analyze retail sales and returns performance across stores, products, and months using an interactive Power BI dashboard.

## Dataset Used
- [Download Dataset](https://github.com/mahmoudnasr33/powerbi-sales-and-returns-analysis/blob/main/DA%20Project1.xlsx)

## Questions (KPIs)

- What are the total sales, total returns, total quantity, and number of stores?
- Which months generate the highest sales?
- Which products generate the highest sales?
- Which stores have the highest and lowest sales performance?
- Which products record the highest and lowest returns?
- How do monthly filters affect sales and returns performance?

## Process

- Cleaned and transformed raw data using Power Query.
- Created measures for total sales, total returns, and total quantity.
- Built a multi-page dashboard (Main, Sales, Returns).
- Designed interactive navigation buttons between report pages.
- Applied slicers for month-based filtering.

## Project Insights

- Total sales reached $21M while total returns reached $2M.
- November recorded the highest monthly sales.
- Caramel Cake generated the highest sales among products.
- Massa Store achieved the highest store performance.
- Fruit Kunafa recorded the highest return value.

## Final Conclusion

The dashboard provides a clear view of both sales and returns performance across products, stores, and months.  
It helps identify top-performing stores, high-return products, and seasonal trends to support better business decisions.
